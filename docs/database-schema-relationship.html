💡 核心业务流程

  1. 通知发布: 创建通知 → 设置接收人 → 添加问题 → 发布
  2. 通知接收: 用户收到 → 记录阅读状态 → 查看问题 → 提交答案
  3. 关系链: 企业微信部门 → 家长学生关系 → 通知接收对象
